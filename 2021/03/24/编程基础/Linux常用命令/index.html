Linux常用命令

发表于 2021-03-24 更新于 2021-09-03 分类于 编程基础

掌握一些基础的Linux命令是非常必要的事情嗷~

ls

显示指定工作目录下之内容, list的缩写.

cd

改变目标文件夹, change directory的缩写.

cp

复制文件或文件夹, copy的缩写.

mv

移动文件或文件夹, move的缩写.

rm

删除文件或者文件夹, remove的缩写.

touch

若文件存在, 则修改文件的时间; 若不存在, 则创建文件.

mkdir

创建文件夹, make directories的缩写.

tree

以树状图列出文件结构.

cat

查看文件内容, 创建文件, 合并文件, 追加文件内容等, concatenate的缩写.

less

分屏显示文件内容.

grep

搜索文件内容, Globally search a Regular Expression and Print的缩写.

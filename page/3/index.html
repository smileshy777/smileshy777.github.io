网络的传递性

发表于 2020-05-21 更新于 2020-05-22 分类于 图算法

传递性(Transitivity)

网络的传递性, 可以用于衡量一个网络中, 各个节点之间相互连通的程度. 在一些网络中, 节点倾向于相互孤立, 那么传递性就较差; 在一些网络中, 节点之间倾向于相互连接, 那么传递性一般就会高一些.

传递性的思想用一句通俗的话来说, 就是: 我朋友的朋友也是我的朋友.

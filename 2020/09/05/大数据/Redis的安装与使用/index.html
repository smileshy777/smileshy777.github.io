Redis的安装与使用

发表于 2020-09-05 更新于 2020-10-10 分类于 大数据

REmote DIctionary Server (Redis) 是一个开源的key - value存储系统, 使用C语言编写, 可基于内存亦可持久化的数据库, 并提供多种语言的API.

数据库简介

Redis 与其他 key-value 缓存产品有以下三个特点：

  • 支持数据的持久化.

    可以将内存中的数据保存在磁盘中, 重启的时候可以再次加载进行使用.

  • 支持多种数据结构.

    不仅仅支持简单的key-value类型的数据, 同时还提供list, set, zset, hash等数据结构的存储.

  • Redis支持数据的备份.

    即master-slave模式的数据备份.

此外, Redis还有如下一些优势:

  • 性能极高.

    Redis能读的速度约100000次/s, 写的速度约80000次/s .

  • 原子性.

    Redis的所有操作都是原子性的.

因此, Redis可以作为一个单独的数据库使用, 也可以配合其它数据库(如MySQL)使用. 可以作为缓存层, 将部分数据存入Redis中, 优先对Redis进行查询, 当查询不到时, 再到全量数据库(如MySQL)中进行查询.

Redis中没有类似MySQL中的数据库, 数据表的结构, 支持5种数据类型:

  • 字符串(String).
  • 哈希(Hash).
  • 列表(List).
  • 集合(Set).
